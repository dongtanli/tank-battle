# 🎮 坦克大战 (Battle City)

经典坦克大战游戏的 HTML5 Canvas 实现，纯前端，打开即玩！

## 在线试玩

直接将 `tank-battle.html` 拖入浏览器，或通过 GitHub Pages 访问。

## 游戏操作

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格 | 射击 |
| P | 暂停/继续 |

## 游戏特色

- 3 个不同关卡（草原、钢铁、水域）
- 砖墙可被子弹破坏，钢墙不可摧毁
- 敌方坦克 AI：随机移动 + 概率追踪玩家
- 粒子爆炸特效
- 基地（鹰旗）保护机制
- 生命系统（3 条命，死后重生）
- 关卡通关后自动进入下一关

## 技术实现

- 纯 HTML5 Canvas API
- 30×30 网格地图系统
- 基于瓦片的碰撞检测
- requestAnimationFrame 游戏循环
- 无任何外部依赖

## 本地运行

```bash
# 直接双击打开
open tank-battle.html

# 或用本地服务器
npx serve .
```
